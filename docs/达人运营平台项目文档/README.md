## 3. 命名规则

后续文档统一按以下格式命名：

```text
版本号-序号-文档主题.md
```

示例：

```text
V1.3.0-00-PRD-数据导出与密码管理.md
V1.3.0-01-需求迭代记录-数据导出.md
V1.3.0-02-测试验收记录.md
```

版本号建议规则：

- `V1.2.1`：当前线上稳定版。
- `V1.3.0`：下一次有明确新功能的版本。
- `V1.3.1`：同一功能版本的小修复。
- 每次上线前，先更新 PRD 和迭代记录，再更新代码版本。
